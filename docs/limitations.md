# Limitations

Version 0.1 is intentionally narrow:

- Local simulation only in v0.1
- Small circuits only
- Educational demos only
- No production quantum workload
- No quantum advantage claimed
- No hardware execution in v0.1
- No noise model in v0.1
- No error mitigation
- No QAOA or VQE
- No pulse-level control
- No quantum error correction
- No industrial-scale optimization

The simulator results should not be interpreted as evidence of useful hardware
performance. Backend execution is optional future work, not part of v0.1.
